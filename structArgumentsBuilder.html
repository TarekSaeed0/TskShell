◆ arguments_builder_build()

Arguments arguments_builder_build ( ArgumentsBuilder *  builder)

Builds the final list of arguments from the arguments builder.

if arguments builder is still in an argument, this discards it. after this, the arguments builder is reset to an empty state, and doesn't need to be dropped. the caller is responsible for dropping the returned Arguments struct when it's no longer needed.

Parameters
[in,out]builderPointer to the arguments builder.
Returns

Ensures that the arguments builder is in an argument.

if the arguments builder wasn't in an argument already, this starts a new argument. this is useful when the caller wants an empty argument, which can't be created by appending characters or strings.
